# Harsith Karne Portfolio Website

A modern, responsive portfolio website showcasing Harsith Karne's academic and professional information.

## Features

- **Modern Design**: Clean, contemporary UI with gradient backgrounds and smooth animations
- **Responsive Layout**: Optimized for all device sizes (desktop, tablet, mobile)
- **Smooth Animations**: CSS animations and JavaScript intersection observer for engaging user experience
- **Bootstrap Integration**: Leverages Bootstrap 5.3.3 for responsive components
- **Custom Typography**: Uses Inter and Montserrat fonts for professional appearance

## Pages

1. **Home** (`index.html`) - Landing page with hero section and call-to-action
2. **About** (`about.html`) - Information about the BUDT748 Industry Practicum course
3. **Contact** (`contact.html`) - Contact details for the course team

## Technologies Used

- HTML5
- CSS3 (with custom properties and modern features)
- Bootstrap 5.3.3
- JavaScript (ES6+)
- Google Fonts (Inter & Montserrat)

## Key Style Improvements

1. **Color Scheme**: Modern gradient backgrounds with CSS custom properties
2. **Typography**: Hierarchical font system with appropriate weights and sizes
3. **Layout**: Flexbox and CSS Grid for responsive layouts
4. **Animations**: Keyframe animations and intersection observer for smooth transitions
5. **Interactive Elements**: Hover effects and button animations
6. **Accessibility**: Proper semantic HTML and ARIA attributes
7. **Performance**: Optimized CSS with efficient selectors

## Browser Compatibility

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## Usage

Simply open any of the HTML files in a modern web browser to view the website. All files are self-contained with external CDN dependencies for Bootstrap and Google Fonts.

For local development, you can serve the files using any static file server or simply open them directly in your browser.
